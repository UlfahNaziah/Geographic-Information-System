Sistem Informasi Geografis – Map Server

1. Latar Belakang

Dalam Sistem Informasi Geografis terdapat fitur aplikasi open source yang biasa disebut dengan Map Server. Untuk lebih jelasnya akan kita bahas sebagai berikut.

1. Pembahasan

1. a)Pengertian Map Server

Map Server merupakan aplikasi open source yang memungkinkan kita bisa menampilkan peta di web. Adanya Map Server mempermudah pekerjaan peta digital menjadi lebih mudah, cepat dan Interaktif. Kata interaktif disini maksudnya pengguna dapat dengan mudah melihat dan mengubah tampilan peta seperti zoom, rotate dan menampilkan informasi dan analisis pada permukaan geografi.

1. b)Fungsi Map Server

Melakukan pembacaan data dari banyak sumber dan menempatkannya kedalam layer-layer secara bersamaan menjadi file graphic.
